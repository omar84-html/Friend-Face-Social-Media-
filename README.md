# FriendFace - Social Media Console App

Et konsollbasert sosialt medium utviklet i C# som en del av GET Academy.

## Funksjoner

- Innlogget som hovedbruker
- Vise alle brukere på plattformen
- Legge til venner
- Fjerne venner
- Vise egen venneliste
- Se profilinformasjon til venner

## Teknologi

- C# (.NET)
- Visual Studio
- Objektorientert programmering (OOP)

## Konsepter brukt

- Klasser og objekter
- Constructor med parametere
- List<T>
- Switch og while-løkke
- for og foreach
- User input med Console.ReadLine

## Hvordan kjøre

1. Åpne prosjektet i Visual Studio
2. Trykk F5
3. Følg menyen i konsollen
